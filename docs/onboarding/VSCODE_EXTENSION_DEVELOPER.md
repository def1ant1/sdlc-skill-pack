# VS Code Extension Developer Onboarding

## Prerequisites
- Open the repository root in VS Code.
- Ensure Python 3 and Node.js 20+ are available.

## Core workflows
1. Compile extension:
   - `npm --prefix extensions/vscode run compile`
2. Manifest validation:
   - Command palette → **Apotheon: Validate Skill Manifests**
3. Workflow dry-run launch:
   - Command palette → **Apotheon: Dry-Run Workflow Launch**
4. Runtime diagnostics:
   - Command palette → **Apotheon: Generate Runtime Diagnostics**
5. Skill maturity report:
   - Command palette → **Apotheon: Skill Maturity Report**
6. Template import:
   - Command palette → **Apotheon: Import OldFarmTrucks Template**

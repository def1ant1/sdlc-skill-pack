# Conversation Extraction

This module defines canonical schemas for extracting structured planning artifacts from conversational transcripts.

## Schema catalog
- `schemas/requirement.schema.json`
- `schemas/risk.schema.json`
- `schemas/decision.schema.json`
- `schemas/task.schema.json`
- `schemas/entity.schema.json`
- `schemas/extraction-result.schema.json`

The aggregate extraction result includes suggestion lifecycle metadata and source message linkage for human-in-the-loop review.

# Job Description Templates

## JD Structure

All job descriptions follow this structure:

```
[ROLE TITLE]
[Department] | [Location/Remote] | [Employment Type] | [Band]

ABOUT APOTHEON
[2-sentence company description]

THE ROLE
[2-3 sentence summary: what the person will do, why this role exists now]

WHAT YOU'LL DO
- [Responsibility 1 — specific, action-oriented]
- [Responsibility 2]
- [Responsibility 3]
- [5–8 bullets total]

WHAT WE'RE LOOKING FOR
Must have:
- [Requirement 1]
- [Requirement 2]
- [3–5 must-haves — specific and measurable]

Nice to have:
- [Preference 1]
- [Preference 2]

COMPENSATION
- Base salary: $[range]
- Equity: [range] over 4-year vest (1-year cliff)
- Benefits: [health, dental, vision, 401K, PTO]

ABOUT OUR PROCESS
[1-2 sentences on interview process and timeline]

Apotheon.ai is an equal opportunity employer. We celebrate diversity and are
committed to creating an inclusive environment for all employees.
```

## Template by Role Family

### Software Engineer

```
WHAT YOU'LL DO
- Design, build, and maintain [component/service]
- Write clean, well-tested code with >80% coverage on domain logic
- Participate in code reviews and improve team code quality standards
- Debug and resolve production issues within SLA
- Collaborate with product and design on feature definition

MUST HAVE (Mid, Band 3-4)
- 3+ years of professional software engineering experience
- Proficiency in [primary language: Go/Python/TypeScript]
- Experience with REST API design and service-oriented architecture
- Strong understanding of data structures, algorithms, and system design fundamentals

NICE TO HAVE
- Experience with AI/ML systems or LLM integrations
- Contributions to open-source projects
- Experience with distributed systems at scale
```

### AI / ML Engineer

```
WHAT YOU'LL DO
- Design and implement LLM-powered features and pipelines
- Build evaluation harnesses and maintain model quality metrics
- Fine-tune and adapt foundation models for production use cases
- Optimize inference cost and latency for AI workloads
- Define prompt engineering standards and version control

MUST HAVE
- 2+ years building production AI/ML systems
- Deep understanding of LLM architectures and capabilities
- Experience with evaluation-driven development (evals, benchmarks)
- Proficiency in Python

NICE TO HAVE
- Experience with LoRA / PEFT fine-tuning
- Familiarity with vLLM, Ollama, or local inference stacks
- Published research or open-source AI contributions
```

### Product Manager

```
WHAT YOU'LL DO
- Own the product roadmap for [area]
- Write clear, testable product requirements and user stories
- Collaborate with engineering and design to ship features on time
- Define success metrics and track outcomes post-launch
- Represent the voice of the customer in all product decisions

MUST HAVE
- 3+ years of product management experience
- Track record of shipping successful B2B SaaS products
- Ability to write detailed requirements with clear acceptance criteria
- Strong analytical skills; comfortable with data and SQL basics

NICE TO HAVE
- Experience in [AI/developer tools/compliance] domain
- Technical background (can read code, understands APIs)
```

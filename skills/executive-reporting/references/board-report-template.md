# Board Report Template

## Board Report Structure

Board reports are 10–15 slides. Lead with the most important item — decisions needed
from the board. Every slide must earn its place.

---

## Slide 1 — Decisions Needed (Always First)

```
DECISIONS NEEDED FROM THE BOARD
================================
1. [Decision question] — Recommendation: [option]
2. [Decision question] — Recommendation: [option]
3. [Decision question] — For awareness only

Context for each: [one sentence per item linking to supporting slide]
```

## Investor Update Format (5–7 slides)

Condensed version for monthly investor updates:

1. **Headline** — MRR + ARR + top win/achievement
2. **Metrics** — Core growth metrics table (MRR, ARR, churn, NRR, burn)
3. **Product** — Features shipped; key metrics
4. **GTM** — Pipeline; signups; conversions
5. **Risks & Asks** — Top risks; what you need from investors
6. *(optional)* **OKR progress**
7. *(optional)* **Appendix** — Detailed financial model

Investor updates are always approved by CEO and CFO before distribution.

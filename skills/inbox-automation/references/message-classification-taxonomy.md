# Inbox Automation — Message Classification Taxonomy

## Intent Classification

| Intent | Description | Routing Behavior | Draft Behavior |
|--------|-------------|-----------------|----------------|
| `request` | Asks for information, action, or decision | Route to domain owner | Draft response for P3/P4 |
| `information` | Provides information with no action needed | Route to relevant team | No draft (FYI only) |
| `escalation` | Flags an urgent issue requiring immediate attention | Route to P1 channel; alert on-call | No draft — route immediately |
| `approval_needed` | Requires a formal decision or sign-off | Route to approver | No draft — human decision |
| `fyi` | Low-urgency informational update | Archive with tag | No action |
| `spam` | Unsolicited, irrelevant message | Archive with spam tag | No action |

## Urgency Classification

```yaml
urgency_rules:
  P1:
    signals:
      - keywords: [urgent, ASAP, critical, down, outage, data loss, breach, immediately]
      - sender_domain_patterns: [incident@*, security@*, ops-alert@*]
      - subject_patterns: ["^\\[P1\\]", "^\\[URGENT\\]", "^ALERT:"]
    response_sla_hours: 1

  P2:
    signals:
      - keywords: [important, priority, today, end of day, EOD, blocker]
      - subject_patterns: ["^\\[P2\\]", "^\\[IMPORTANT\\]"]
    response_sla_hours: 4

  P3:
    signals:
      - keywords: [please, when you get a chance, by end of week, question]
    response_sla_hours: 24

  P4:
    signals:
      - default: true  # No urgency signals detected
    response_sla_hours: 72
```

## Classification Confidence Thresholds

```yaml
confidence_thresholds:
  autonomous_routing:
    minimum: 0.75           # Route autonomously only if confidence >= 75%
    below_threshold_action: "Create routing task for human reviewer"

  draft_generation:
    minimum: 0.80           # Generate draft only if high confidence in intent+domain
    below_threshold_action: "Route without draft; add 'Manual draft needed' flag"

  spam_classification:
    minimum: 0.90           # Very high confidence required before marking as spam
    below_threshold_action: "Route to low-priority queue for human review"
```
